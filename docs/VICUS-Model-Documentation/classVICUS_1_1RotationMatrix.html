VICUS::RotationMatrix Class Reference

Detailed Description

This is a pretty dumb class and only used for the serialization code generator.

It stores 4 floats and matches internally the QQuaternion class. When we copy back and forth from a QQuaternion, we just treat ourselves as a QQuaternion class (low level memory access is can be beautiful :-)
