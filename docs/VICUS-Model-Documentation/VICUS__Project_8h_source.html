137 /*! Searches through all objects and determines the largest object ID (not unique ID!) used for buildings, buildingLevels, rooms, surface,
138 subsurfaces, networks, etc and returns the next ID to be used for new data elements. For example, if IDs 10, 11, 14 have been used already,
139 the function returns 15.
140 \note Expects that updatePointers() has been called beforehand (i.e. project is synced)
141 */

144 /*! Searches through all objects in project structure for the object with the object ID.

151 \return Returns nullptr, when object cannot be found.

155 }
