VICUS::Polygon3D Class Reference

Detailed Description

Class Polygon3D stores a planar polygon in 3D space.

This class merely wraps IBKMK::Polygon3D and provides read/write functionality.

inline

Constructs a polygon from a 3D polyline (which might be invalid in any number of ways).

The normal vector will be deduced from rotation direction of the polygon, and the x-axis vector will be the vector from first to second vertex at a suitable (automatically selected) vertex of the polygon.

Note
Once all collinear points have been removed the offset point will be the first vertex of the polygon. Use offset() to retrieve the offset.
