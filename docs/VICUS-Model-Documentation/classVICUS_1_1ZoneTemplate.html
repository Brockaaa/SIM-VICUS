◆ usedReference()

ZoneTemplate::SubTemplateType VICUS::ZoneTemplate::usedReference ( unsigned int  index) const

Returns the type of reference by index, counting only the used references, i.e.

references not INVALID_ID. For example, if m_idReferences[ST_IntLoadPerson] == INVALID_ID and m_idReferences[ST_IntLoadEquipment] has a valid ID, than usedReference(0) returns ST_IntLoadEquipment.

Returns
Returns type of corresponding id reference or NUM_ST, if index is larger than the number of non-empty id references.
