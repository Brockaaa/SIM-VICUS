VICUS::Database< T > Class Template Reference

Detailed Description

template<typename T>
class VICUS::Database< T >

Wrapper class for different databases.

Template argument classes must provide a public bool member m_builtIn, and the functions readXML(const TiXmlElement * element) and writeXML(TiXmlElement * parent) const.

Usage:

VICUS::Database<Material> materials;
VICUS::Material m;
