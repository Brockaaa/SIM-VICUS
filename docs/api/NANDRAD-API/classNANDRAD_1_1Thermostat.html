NANDRAD::Thermostat Class Reference

Detailed Description

Contains data for a thermostat model, that compares the air/operative tempeature with a setpoint and generates either an analog or digital control signal.

The latter always uses hysteresis.

◆ ControllerType

enum NANDRAD::Thermostat::ControllerType
Enumerator
CT_Analog 

Signal is the difference between setpoint temperature and sensor temperature, normalized by the given temperature tolerance.

A signal value of 1 means that the temperature tolerance has been reached, a value larger than 1 means the temperature difference is even larger (signal value is not clipped).

CT_Digital 

Signal is either 0 or 1, and corresponds with the digital hysteretic control logic.

1 is returned if heating/cooling is required.
