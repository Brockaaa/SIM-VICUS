NANDRAD::EmbeddedObject Class Reference

Detailed Description

Declaration for class EmbeddedObject.

An embedded object generally defines a wall opening (a window or a door). That means, the calculation radiant heat fluxes and heat fluxes by heat transmission are performed by a window model or door model. The embedded oject stores an exchangable parameter model that defines the name for the calculation model and constant model parameters.
