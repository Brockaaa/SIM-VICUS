Keyboard and Mouse Controls

Normal Scene Navigation

Click and hold left mouse button

Orbit controller (camera rotates around point where mouse button was clicked)

Hold right mouse button

First person controller (mouse rotates camera around camera position)

Hold middle mouse button

Mouse translates camera

Scroll wheel

translate camera forward/backwards (hold Shift-Key: slow down translation)

W, S

translate camera forward/backwards (hold Shift-Key: slow down translation)

A, D

translate camera left/right respective to camera view (hold Shift-Key: slow down translation)

R, F

translate camera up/down respective to camera view (hold Shift-Key: slow down translation)

Q, E

rotate camera around z-axis (hold Shift-Key: slow down translation)

Ctrl + 2,6,8,4 or 5

Reset camera to look from north, east, south, west or from above (use numpad!)

Item Selections

Left button click

Select/deselect surfaces (hold Shift-Key: also select parent and all siblings, hold Alt-Key: only select object itself and not its children)

Escape

Delesects all selected surfaces.

Ctrl+F

Locates/finds the selected geometry, i.e. moves close to the local coordinate system. Useful when selecting an item from the navigation tree view and then quickly locating this element in the geometry.

While drawing new geometry

Controls work as for normal scene navigation, with the changes described below:

Left button click

Place a vertex at the current position of the local coordinate system.

Escape

Cancels drawing operation and goes back to normal scene navigation mode.

Enter

Once a valid geometry is drawn (and shown), this geometry will be added to the project and drawing is finished.

Global shortcuts

These shortcuts/keys work globally, but some only in a certain context, for example when the local coordinate system (LCS) is shown.

F1

Opens this quick reference page.

F2

Opens current project file in text editor.

F3

Toggle snapping on/off.

F4

Put scene into "align local coordinate system" mode (only when LCS is shown).

F5

Put scene into "move local coordinate system" mode (only when LCS is shown).

F6

Hide selected objects.

F7

Show selected objects.

F8

Opens directory of current project file in file manager.

F9

Switch to simulation start dialog/view.

c

Toggle between geometry editing and parametrization mode.

x,y,z

While drawing/creating geometry/moving local coordinate system: lock selected axis.

0...9 and -

While creating geometry, open an edit field to enter coordinate offsets.
